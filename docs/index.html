Project Mate Deep Dive

The hero workflow: from an idea to verified impact.

Project Mate makes team formation and execution easy to explain in a live demo.

01 · RECRUIT

Role-aware teammate discovery

Owners publish a project, define slots and capacity, then review applicants for each open role.

02 · COORDINATE

Private workspace control center

Accepted members use coordination settings, team discussion, shared resources, and a teammate console.

03 · DELIVER

Task accountability

Leads assign deadlines, members submit deliverables, and extension requests stay understandable.

04 · VERIFY

Contribution proof

Verified work flows into profile portfolio sections and a clearer contribution history.

Security

Public-safe and workspace-aware.

  • Supabase Row Level Security protects data access.
  • Private files use signed, time-limited URLs.
  • Raw storage paths stay hidden from the UI.
  • Contact details remain private unless allowed.
Tech Stack · Architecture

Lean, modern, deployment-ready.

  • React 19 + TypeScript + Vite + Tailwind CSS
  • Supabase Auth + PostgreSQL + Storage + RLS
  • Feature pages, reusable components, typed service helpers
  • Vercel live app + static GitHub Pages showcase
Setup · Deployment

Simple to run and simple to present.

Local setup

  • Install dependencies with npm install.
  • Copy .env.example to .env.local.
  • Add frontend-safe Supabase URL and anon key values.
  • Start with npm run dev.

Deployment

  • Build with npm run build.
  • Deploy the Vite output directory dist to Vercel.
  • Publish this static showcase from /docs on GitHub Pages.
  • Keep secrets out of source control.
